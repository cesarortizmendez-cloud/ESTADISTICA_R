[⇄] Transformador CSV → Tablas

Sube un archivo y obtén, en un clic, su tabla de frecuencias lista para Excel.

Carga un .csv o .xlsx. El proceso lo lee como un data frame de R, lo describe y construye automáticamente una tabla de frecuencias. Si la columna elegida es numérica, aplica la regla de Sturges (K = 1 + 3.3·log₁₀(n)) y arma intervalos; si es categórica, cuenta cada categoría. Todo corre con R real en tu navegador.

▸ Cómo funciona por dentro

  1. El archivo se lee con SheetJS y se escribe en el sistema de archivos de R como CSV.
  2. R lo importa con read.csv() a un data frame llamado tabla.
  3. Para columnas numéricas: número de clases con Sturges, cut() para los intervalos y table() para contar.
  4. Se calculan frecuencia absoluta, relativa, porcentual y acumuladas en un data frame frecuencias, exportable a Excel.
